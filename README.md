# E-Commerce Management System

## Project Description
This is a simple **E-Commerce Management System** developed using Java, JDBC, and MySQL.

It helps manage users and products with basic shopping operations.

---

##  Features

- User Registration  
- User Login  
- Admin Login  
- Add Product  
- View Products  
- Update Product  
- Delete Product  
- Database Connectivity (JDBC)

---

##  Tech Stack

- Java (Core Java)
- JDBC
- MySQL
- Eclipse IDE

---

##  How to Run

1. Clone the repository  
2. Open project in Eclipse  
3. Setup MySQL database  
4. Update DB credentials in `DBConnection.java`  
5. Run `Main.java`
